OVERVIEW
Queue Detector

Queue detection module based on a neural network. Includes a reporting module and alerts on exceeding the queue length. When used in conjunction with the TRASSIR ActivePOS module, it allows you to enable alerts based on the number of operating cash registers. Works on the Neuro Station server. Offload analytics is possible (multiple RSSTEK servers transmit images to 1 server for processing).

How it Works

  • Monitor queue lengths;
  • Respond to incidents in a timely manner;
  • Use statistics to optimize work processes.

The functionality can be expanded by connecting additional scripts.
